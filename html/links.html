List of cool software

I figured I'd put a list of software I like or enjoy here and not make a blogpost as it's neater. NOTE that I mainly use Flatpak with Flathub nowadays, unless I'm on an OS that is based on Arch. Also see the software I tend to use in About.

Category Cool software
Operating Systems

Fedora Linux

Fedora is a Linux distribution and prides itself on adopting the latest features and accomplishments of Linux, like Flatpak, Btrfs and Wayland. +2 for Fedora Silverblue/Kiniote

EndeavourOS

EndeavourOS is an Arch-based Linux distribution that allows you to setup a preconfigured Arch installation with whatever desktop environment you want. Basically, Arch with a GUI installer.
Desktop Environments

GNOME

GNOME is a Linux desktop environment focused on keeping things simple so you can just get work done. Made in GTK.

KDE Plasma

KDE Plasma is a Linux desktop environment and software ecosystem where the cool is everything is customizable. Made in Qt.
File Managers

Dolphin

Dolphin is part of KDE and is the most powerful file manager I've used. Tab support, split view and limitless customisability. Everything else on linux just lacks.
Web Browsers

Firefox

Firefox is the best browser out there, use it. By default, it is reasonably secure and private. Read more with my Firefox Guide

LibreWolf

LibreWolf is a fork of Firefox aimed at stripping out all telemetry and tracking, and hardening the browser against tracking, fingerprinting and malware as much as possible. Useful if you want a hardened browser but don't want to deal with TOR.

TOR Browser

TOR Browser is a modified version of Firefox with TOR included and a lot of specific tweaks on top and is generally the recommended way to stay completely anonymous when browsing stuff. Can be slow, since you're using Tor.

Thorium

Thorium is a fork of Chromium with a bunch of tweaks applied to make it faster and is more private than vanilla Chromium. If you need to use a secondary Chromium browser, use this or Ungoogled Chromium. This is on Linux but no Flatpak.
Audio/Music Players

Elisa

Elisa is a KDE app that plays music. I think this is the best music player I've seen on Linux so far. Very fast and easy to understand.

Amberol

Amberol is what I use when I need to use GNOME. Very simple app but works perfectly if all you need to is just listen to music.

foobar2000

foobar2000 is the best music player that I've used, although it does look ugly. Windows only.
Video Players

mpv

mpv is the best video editor out there, miles better than VLC. Lean and fast and the configuration is practically endless.

Celluloid

Celluloid is a GTK frontend for mpv.
