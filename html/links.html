This is a list of different website I find cool or useful.

Computers

Games

  • Emulation General Wiki - Wiki for all information involving emulators and what emulators to use for different systems.
  • r/Roms Megathread - Best place to look for ROMs.
  • PCGamingWiki - Wiki that documents details about PC games and common errors to troubleshoot.

Linux

  • Arch Wiki - mainly documentation for Arch Linux, but is extremely useful for any distro and even Windows, and is one of the best wikis out there.
  • pkgs.org - Want to see if a piece of software is available in a distro's repos? use this tool.

General computer stuff

  • AlternativeTo - Find alternatives to all the programs you use.
  • Neocities - Spiritual successor to geocities, people can make their own little personal websites there.

Computer Privacy & Security

  • Terms of Service; Didn't Read - Search a website and see the privacy and security grade of it.
  • PrivacyGuides - Sane guide to privacy (and security) on the internet.
  • have i been pwned? - Has your email address been in a data breach? check here.
  • VirusTotal - Are you unsure of a file you downloaded? This tool lets you upload it and checks like 50 different antiviruses on it.

  • no hello - Don't say hello, just ask the question.
  • TinEye - Good reverse image search to check if images might've been altered.

List of cool software

I figured I'd put a list of software I like or enjoy here and not make a blogpost as it's neater. NOTE that I mainly use Flatpak with Flathub nowadays, unless I'm on an OS that is based on Arch. Also see the software I tend to use in About.

Category Cool software
Operating Systems

Fedora Linux

Fedora is a Linux distribution and prides itself on adopting the latest features and accomplishments of Linux, like Flatpak, Btrfs and Wayland. +2 for Fedora Silverblue/Kiniote

EndeavourOS

EndeavourOS is an Arch-based Linux distribution that allows you to setup a preconfigured Arch installation with whatever desktop environment you want. Basically, Arch with a GUI installer.
Desktop Environments

GNOME

GNOME is a Linux desktop environment focused on keeping things simple so you can just get work done. Made in GTK.

KDE Plasma

KDE Plasma is a Linux desktop environment and software ecosystem where the cool is everything is customizable. Made in Qt.
File Managers

Dolphin

Dolphin is part of KDE and is the most powerful file manager I've used. Tab support, split view and limitless customisability. Everything else on linux just lacks.
Web Browsers

Firefox

Firefox is the best browser out there, use it. By default, it is reasonably secure and private. Read more with my Firefox Guide

LibreWolf

LibreWolf is a fork of Firefox aimed at stripping out all telemetry and tracking, and hardening the browser against tracking, fingerprinting and malware as much as possible. Useful if you want a hardened browser but don't want to deal with TOR.

TOR Browser

TOR Browser is a modified version of Firefox with TOR included and a lot of specific tweaks on top and is generally the recommended way to stay completely anonymous when browsing stuff. Can be slow, since you're using Tor.

Thorium

Thorium is a fork of Chromium with a bunch of tweaks applied to make it faster and is more private than vanilla Chromium. If you need to use a secondary Chromium browser, use this or Ungoogled Chromium. This is on Linux but no Flatpak.
Music Players

Elisa

Elisa is a KDE app that plays music. I think this is the best music player I've seen on Linux so far. Very fast and easy to understand.

Amberol

Amberol is what I use when I need to use GNOME. Very simple app but works perfectly if all you need to is just listen to music.

foobar2000

foobar2000 is the best music player that I've used, although it does look ugly. Windows only.
Video Players

mpv

mpv is the best video editor out there, miles better than VLC. Lean and fast and the configuration is practically endless.

Celluloid

Celluloid is a GTK frontend for mpv.
Image Editors

GIMP

GIMP is practically and open source version of Photoshop that has nearly all the features that PS offers, and it's free.

Krita

Krita is part of the KDE suite and is very popular for raster graphics. I mainly use this to draw boxes because i suck at art

Aseprite

Aseprite is the best tool I've used to make pixel art. You can either buy it or compile it yourself. Source code is here. Has Linux version, no flatpak. Also check LibreSprite.
Office

LibreOffice

LibreOffice is the gold standard for open source office suites. Has Writer (Word), Calc (Excel), Impress (PowerPoint), Draw (Visio/Photoshop), Math and Base (Access).

OnlyOffice

OnlyOffice looks like a ripoff version of Microsoft Office, practically an Open Source version of Office. Has better compatibility with Office formats than LibreOffice.
Code Editors

Visual Studio Code

VSCode is made by Microsoft and is the best editor I've used. Extensions and integrated GitHub is way too good to switch to anything else. The base is open source but the binaries are proprietary.

Kate

Kate is part of the KDE suite and is a very nice code editor/text editor packed full of features. Not as powerful as VSCode but I tend to use it when editing files quickly because it is faster.
Password Managers

KeePassXC

KeePassXC is the only password manager I recommend. It stores an encrypted password file locally and you can sync it with an external program or cloud. Browser extension exists for autofill integration. Has 128-bit password generation.
