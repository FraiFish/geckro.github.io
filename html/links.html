This is a list of different website I find cool or useful.

  • Emulation General Wiki - Wiki for all information involving emulators and wha emulators to use for different systems.
  • r/Roms Megathread - Best place to look for ROMs.
  • Terms of Service Didn't Read - Search a website and see the privacy and security grade of it.
  • SuperCook - Allows you to select ingredients and then it'll spit out recipes based on the ingredients you have.
  • have i been pwned? - Has your email address been in a data breach? check here.
  • VirusTotal - Are you unsure of a file you downloaded? This tool lets you upload it and checks like 50 different antiviruses on it.
  • no hello - Don't say hello, just ask the question.
  • TinEye - Good reverse image search to check if images might've been altered.
  • TheNounProject - Free icons for projects.
  • AlternativeTo - Find alternatives to all the programs you use.
  • PrivacyGuides - Sane guide to privacy (and security) on the internet.

List of cool software

I figured I'd put a list of software I like or enjoy here and not make a blogpost as it's neater.

Note that I mainly use Flatpak with Flathub nowadays, unless I'm on an OS that is based on Arch. Apps that are orange are integrated into GNOME and apps that are blue are integrated into KDE Plasma.

Also see the software I tend to use in About.

W.I.P.

Category Cool software
Operating Systems Fedora / Fedora Silverblue / EndeavourOS
Desktop Environments GNOME / KDE Plasma
Display servers Wayland
File Managers Dolphin

Dolphin is simply the most powerful file manager out there in my experience. Tabbed folders, split view, fast searching, and more; and it's all as customizable as you want. Everything else just lacks in comparison.
Web Browsers Firefox / Vivaldi / TOR Browser
Audio/Music Players Amberol / Elisa

TUI Editors nano

Password Managers Secrets / KeePassXC
Torrentors Fragments / qBitTorrent / Transmission
Download managers KGet

KGet is the only download manager that competes with JDownloader. Has a great UI, tons of options. Linux only.
JDownloader

I really only recommend this because it's sort of a "it's the best one" situation. This program kinda sucks ass and I really hope there's something different that has all or most of its capabilities.
Video Editors Kdenlive / DaVinci Resolve
Image Editors GIMP
Email clients Geary
Office Suites LibreOffice / OnlyOffice
Terminals Konsole / Alacritty
Virtualisation GNOME Boxes

GNOME Boxes is the best virtualization tool for testing out linux distros inside Linux. You can automatically install Fedora, Debian, Windows, Ubuntu and more with one click.
VirtualBox

VirtualBox is my recommended "general use" virtualisation software as its available on everything is understandable, is packed with features and does not come with a fee (vmware). No Flatpak, you have to install this via normal distro repos.
PDF / Document Viewers Okular

Okular is part of KDE and is the most powerful document viewer that I know of. Allows PDF files, comics, books, images, markdown and can edit PDF files.
Other Blanket

Blanket is an application that lets you listen to various types of white noise, and even at the same time. Stuff like wind, rain, coffee shops, storms etc.
Flatseal

Flatseal allows you to manage the permissions of a Flatpak application.
Kooha

Kooha is a small utility with no fluff that allows you to record your screen/audio. I have had no issues at all with it. Exports in webm, gif, mp4.
KTeaTimer

KTeaTimer is literally a timer for you to use when brewing tea. its kinda funny
