Software List

This is all the software I tend to use for these categories.

Basics

  • Operating system: Windows 11 for Windows, and EndeavourOS or Fedora for Linux.
  • Web browser: Firefox, or Vivaldi/Edge/Thorium for Chromium browsers.
  • Terminal Emulator: Konsole
  • File explorer: Whatever comes by default

Productivity

  • Calendar: Google Calendar
  • Cloud storage: OneDrive/Google Drive
  • Email: Gmail/Outlook/Proton
  • Notetaking: Notion.

Multimedia

  • Music player: MusicBee or foobar2000 for Windows, Strawberry for Linux.
  • Audio tagging: MusicBee/foobar2000 inbuilt tagger or mp3tag.
  • Video player: mpv.
  • Image viewer: Whatever comes with the system or desktop environment by default.
  • Image editor: GIMP or paint.NET.
  • Pixel art: Aseprite.
  • Painting program: Krita.
  • Torrent client: qBitTorrent

Programming

  • Text editor: VSCode, or Kate
  • IDE: Whatever JetBrains has

Security

  • Password manager: Bitwarden.

Video gaming (epic)

  • Minecraft: Prism Launcher
  • Game Launchers: Steam/Emulators
