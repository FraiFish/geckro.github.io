Guide to [yt-dlp]

Date created: 2023-04-12 | Last updated: 2023-04-12 | ID: 005

Installation

Windows

Download from here. Place the executable in it's own folder somewhere like USER/Documents/yt-dlp/yt-dlp.exe. When you want to run yt-dlp, open a terminal and navigate to that folder, and then use ./yt-dlp.exe (args) (url)

MacOS

Use brew install yt-dlp. Run yt-dlp by opening a terminal and running yt-dlp (args) (url)

Linux

Check your package manager. More information here.

Basic Usage

To download a video, use yt-dlp [url] (without the brackets)

By default, yt-dlp downloads the best available quality it can.

Formats

Using -f, you can specify the format you want. You can use 3gp, aac, flv, m4a, mp3, mp4, ogg, wav, webm. By default yt-dlp picks the best quality for you.

Extracting audio

Using -x, you can make yt-dlp extract the audio from a video file. By default it extracts the best audio format but you can change it with --remux-video FORMAT.

Available audio FORMATs: avi, flv, gif, mkv, mov, mp4, webm, aac, aiff, alac, flac, m4a, mka, mp3, ogg, opus, vorbis, wav
