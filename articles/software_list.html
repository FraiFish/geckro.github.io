Operating system

Windows 11 for Windows-based operating systems, and EndeavourOS or Fedora for Linux-based operating systems.

Web browser

Thorium for Chromium browsers, or Firefox if you don't want Chromium/Blink.

Terminal emulator

Konsole for Linux, and Windows Terminal for Windows.

Music player

MusicBee for Windows, Strawberry for Linux.

Video player

mpv. If for some reason mpv doesn't work (which is usually never), just use VLC.

Text editors / IDEs

VSCode. For Python and .NET I use JetBrains PyCharm and Rider respectively.

Password manager

Bitwarden. If you don't want to rely on Bitwarden's servers and cannot self-host, use KeePassXC.

Image editors

GIMP is good enough for what I need to do. For pixel art, I paid for Aseprite.

Digital art

Krita doesn't cost $5 trillion dollars and a sacrifice to get the software so I use it

Notetaking

I use Notion. If you don't want to have your notes online, try out Obsidian.

Email

I use Gmail, on a browser. Email is not important to me at all to really protect and email clients are super annoying to use. I also use Google Calendar, Drive, Maps, Photos and such.

Emulators

I only emulate games on my desktop.

  • 32X: RetroArch (PicoDrive)
  • 3DS: Citra (fork)
  • Dreamcast: Flycast
  • DS/i: RetroArch (melonDS) or melonDS (Standalone)
  • GameCube / Wii: Dolphin
  • Game Boy / Color: RetroArch (SameBoy or Gambatte)
  • Game Boy Advance: RetroArch (mGBA) or mGBA (Standalone)
  • Genesis/Master System/Game Gear/Sega CD: RetroArch (GenesisPlusGX)
  • N64: RetroArch (Mupen64), or Rosalie's Mupen GUI
  • NES: RetroArch (Mesen)
  • PS1: DuckStation, PS2: PCSX2, PS3: RPCS3, PSP: PPSSPP, PSVita: Vita3K
  • Saturn: RetroArch (Beetle Saturn)
  • SNES: RetroArch (Snes9x)
  • Switch: Ryujinx
  • Wii U: Cemu
  • Xbox: Xemu, Xbox 360: Xenia or Xenia Canary
