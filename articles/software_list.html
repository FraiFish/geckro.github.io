Operating system

Windows 11 for Windows-based operating systems, and EndeavourOS or Fedora for Linux-based operating systems. I am using Windows 11 at the moment.

Web browser

Thorium for Chromium browsers, or Firefox if you don't want Chromium/Blink. Currently I'm on Thorium.

For mobile phones, I recommend Firefox on mobile or Cromite (active fork of Bromite).

I recommend the following extensions:

  • uBlock Origin - This is the only ad/content blocker you should use.
  • SponsorBlock - Allows you to skip sponsored segments of videos, and more (filler segments, non music sections)
  • Violentmonkey - Userscript manager, open source version of Tampermonkey and updated version of Greasemonkey.
  • Stylus - Custom CSS on websites. Useful to make dark-mode websites light-mode.

Terminal emulator

Konsole for Linux, and Windows Terminal for Windows.

Multimedia üé∂

Music player

MusicBee for Windows, and Strawberry for Linux. I'm not a fan of music players that are extremely simple.

Video player

mpv. If for some reason mpv doesn't work (which is usually never), just use VLC.

Image editors

GIMP is good enough for what I need to do, although it really needs a revamp (and a much better name). For pixel art, I paid for Aseprite.

Digital art

Krita doesn't cost $5 trillion dollars and a sacrifice to get the software so I use it :p

I also use OpenTabletDriver for my Wacom tablet.

Other üåä

Text editors / IDEs

VSCode. For Python and .NET I use JetBrains PyCharm and Rider respectively.

Password manager

Bitwarden. If you don't want to rely on Bitwarden's servers and cannot self-host, use KeePassXC.

Notetaking

I use Notion. If you don't want to have your notes online, try out Obsidian.

BitTorrent client

I rarely actually torrent nowadays, but when i need Linux ISOs i use qBittorrent.

Office suite

LibreOffice should be fine for most people, but you can just use the Google suite, or if you really need the complicated MS stuff then just use 365.

Email

I use Gmail, on a browser. Email is not important to me at all to really protect and email clients are super annoying to use for me. I also use Google Calendar, Drive, Maps, Photos and such.

Launchers

For the most part, I just use Steam. Sometimes I end up making a Playnite library to consolidate all my launchers, but it usually gets abandoned. For Minecraft, I exclusively use Prism Launcher.

Emulators

I only emulate games on my desktop.

  • 32X: RetroArch (PicoDrive)
  • 3DS: Citra (fork)
  • Dreamcast: Flycast
  • DS/i: RetroArch (melonDS) or melonDS (Standalone)
  • GameCube / Wii: Dolphin
  • Game Boy / Color: RetroArch (SameBoy or Gambatte)
  • Game Boy Advance: RetroArch (mGBA) or mGBA (Standalone)
  • Genesis/Master System/Game Gear/Sega CD: RetroArch (GenesisPlusGX)
  • N64: RetroArch (Mupen64), or Rosalie's Mupen GUI, or Parallel Launcher for ROM hacks
  • NES: RetroArch (Mesen)
  • PS1: DuckStation, PS2: PCSX2, PS3: RPCS3, PSP: PPSSPP, PSVita: Vita3K
  • Saturn: RetroArch (Beetle Saturn)
  • SNES: RetroArch (Snes9x)
  • Switch: Ryujinx
  • Wii U: Cemu
  • Xbox: Xemu, Xbox 360: Xenia or Xenia Canary
