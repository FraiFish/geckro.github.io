001 | Firefox Configuration Guide

Last updated: 2023-04-01

There are a lot of Firefox configuration guides out there but this is how I like to set up Firefox. I plan on making Firefox faster, enhance its security & privacy, add and remove features and try to explain why I tweak each setting.

Disclaimer

This guide assumes you have a tiny bit of knowledge of general computing stuff. I am not responsible for breaking any software on your PC.

Why use Firefox? 🌐

While you may be content using your current browser, whether that be Chrome, Edge or Opera, why not settle for a better browser? With Firefox, if you want to have a browser that supports unparalleled customizibility, near top-of-the-line privacy and security with no strings attached and without unnecessary features, Firefox gives that to you and is a great option. Also, the source code is publically viewable here so you know that your data is kept private, secure and away from other eyes.

But that's not all Firefox offers. With features like a password manager, device syncing, an in-built screenshot tool, and an in-built blocker by default with no extension needed AND more, Firefox may be a good browser to switch to and keep for the near future.

Why should you not use Firefox? ❌

If you use websites often that need the blink engine, they will not work or be very slow in Firefox and if you have a really old computer with less than 4GB of RAM, Firefox may be a too resource intensive and therefore, slow for you. Thankfully, 99.9% of all websites you come across should work fine in Firefox.

Pre-Setup 🛠️

You can download Firefox from here. If you like to get new features early with a stability tradeoff, you can download different versions here .The difference between Beta, Developer and Nightly is how frequently they update it. Beta gets updated every month, Developer gets updated every week or so and Nightly is an update nearly every day.

If you already have Firefox installed and don't use it, i recommend you refresh it by clicking on the ≡ button (top right), click Help and click More troubleshooting information and then click Refresh Firefox.... This saves your bookmarks, passwords and history but removes your extension, modified settings and any customizations you made.

Configuration Guide ⚙️

Click the ≡ button (top right) and click Settings. At the left are the setting categories. You're currently in the General category.

✅ - Turn on

🔘 - Personal Preference

❌ - Turn off

To see settings click on a category.

🔨 General

Startup

🔘 Open previous windows and tabs

Up to personal preference. When you close Firefox with tabs open and launch Firefox, those tabs are automatically loaded.

✅ Always check if Firefox is your default browser

Windows 10/11 tends to replace your default browser with Microsoft Edge after updates, so checking this is a good idea. Also, make Firefox your default browser if you want.

Tabs

🔘 Ctrl+Tab cycles through tabs in recently used order

Up to personal preference. Instead of cycling through it left to right by default, ctrl+tab cycles through tabs by last used.

🔘 Open links in tabs instead of new windows

Up to personal preference. When you click a link like this, it opens in a new browser window.

🔘 When you open a link, image or media in a new tab, switch to it immediately

When you open any link or image, it opens in a new tab and you switch to that tab automatically.

🔘 Confirm before closing multiple tabs

When you have multiple tabs open and click X, Firefox asks if you want to close all the tabs. Good if you're a tab hoarder.

🔘 Show tab previews in the Windows taskbar

When you hover over the Firefox icon, you can see tabs like different windows.

Website appearance

✅ Automatic

This makes it so if you're using light mode on your system, websites will adapt to light mode. If you're dark mode on your system, websites will adapt to dark mode.

Colors

❌ Manage Colors...

It's best if you leave this alone but if you really want to customize it you can.

Fonts

❌ Default font

Like colors, it's best if you leave this alone but if you really want change the monospace font to your favourite, you can here.

Zoom

✅ Default zoom 100%

You can change the zoom of websites by ctrl+scroll wheel

🔘 Zoom text only

This one is hard to explain. Turn it on and then ctrl+scroll wheel in the settings menu to see if you like it or not.

Language

🔘 English (US)

If you speak a different language, put it here otherwise if you're English then just leave it. If you speak British English you can set it here if it isn't set already.

🔘 Use your operating system settings for "LANGUAGE" to format dates, times, numbers, and measurements.

This formats your dates, times etc. by your country defaults. i.e. Australia uses dd/mm/yyyy, metric system and the 12 hour clock.

✅ Check your spelling as you type

This turns on a dictionary that does not connect to the internet or trasmit data to a server. No reason to not have this on unless you have a slow PC or don't like spellcheck.

Downloads

🔘 Save files to 📥 Downloads

If you like to save all your files to a different drive, like D:\Downloads, you can save it here.

🔘 Always ask you where to save files

Every time you download something, it prompts you to where you want to save it.

Applications

🔘 Handle Files

It's best to leave this as default. If you want WebP and AVIF images to save to your downloads instead of opening it, you can change it here.

🔘 What should Firefox do with other files?

Save files just saves files with no fuss. The other option asks whether to open that file type or save that file type.

Digital Rights Management (DRM) Content

🔘 Play DRM-controlled content

If you want to watch Netflix or any paid streaming platform, leave this ✅ enabled. If you are against the idea of DRM, turn this off. You can learn more about DRM here.

Updates

✅ Automatically install updates

This automatically installs updates for you with no fuss. If you really dont want your browser to update to new versions unless you tell it to, you can change it, but not updating your web browser exposes you to many security threats.

Performance

✅ Use recommended performance settings

This checks your hardware and tailors performance settings for you. If you need to disable GPU acceleration, uncheck this and uncheck hardware acceleration.

Browsing

✅ Use autoscrolling

When you middle click your mouse, this allows autoscrolling. Turn off if you hate this for whatever reason.

🔘 Use smooth scrolling

If on, it uses a smooth animation to scroll. If not on, it doesn't. Try it out for yourself.

✅ Show a touch keyboard when necessary

This may help if you need to search something but don't have a keyboard plugged in.

❌ Always use the cursor keys to navigate within pages

This will break a lot of websites, especially games that use the arrow keys.

❌ Search for text when you start typing

Instead of using Ctrl+F, this automatically searches for text in a webpage once you press any key. This will break way too many websites.

🔘 Enable picture-in-picture video controls

If you use and like PiP, keep it. if you don't, disable this. PiP pops videos out webpages so you can watch a YouTube video while playing a game or something.

✅ Control media via keyboard/headset/virtual interface

If you have media keys on your keyboard, enable this. If you don't, keep this on anyway. it wont hurt you.

❌ Recommend extensions as you browse

When you browse websites Firefox recommends extensions for those sites when you browse. Firefox does not receive this data or share or sell this data with third-parties, but it is annoying.

❌ Recommend features as you browse

Read above.

Network Settings

❌ Enable DNS over HTTPS

DoH is a legitimate security feature but i recommend you do this in your system settings instead. In Windows 11 it is in Settings > Network & Internet > Wi-Fi > Hardware properties > Edit DNS server assignment > Manual and change the DNS here. I recommend Cloudflare (1.1.1.1)

🏠 Home

New Windows and Tabs

🔘 Homepage and new windows & New tabs

Up to personal preference. if you click Blank Page it will literally give you a blank page. you will need to search from the address bar.

Firefox Home Content

❌ Sponsored Shortcuts

Turn this off, no reason to have random websites you don't visit in your Home page.

🔍 Search

Search Bar

🔘 Address bar or Address Bar + Search Bar

Up to personal preference. If you like having a seperate search bar you can enable it here. There is an advanced setting called keyword.enabled in about:config that makes the address bar unsearchable.

Default Search Engine

🔘 I recommend DuckDuckGo

If you don't want to use Google, DDG is the best choice. You can also use Bing here. If you want to add other search engines, go to the website (i.e. startpage.com), click the address bar and you should see an option in the very bottom to add that website.

Provide Search Suggestions

🔘 If you care about privacy, turn these off.

This turns off the feature on when you type into the search bar/address bar, it shows you what to type. In order for Firefox to get you these results it contacts your search engine.

❌ Show search suggestions in Private Windows

Private mode should really be a private mode.

🔐 Privacy & Security

Enhanced Tracking Protection

✅ Strict

This does not break most websites and only helps to protect security. If you need to turn off tracking protection look at the top left of the address bar. You can turn it off per website there.

❌ Do Not Track

Do Not Track is not what it says. It basically sends a request to every website you visit and the website can choose to honour that request or not. And most of the time, they don't. It's just additional tracking information.

Cookies and Site Data

🔘 Delete cookies and site data when Firefox is closed

Cookies are still used to track you, but if you clear cookies then you will need to log-in to every website unless you make exceptions to those websites. It's not really worth it.

Cookie Banner Reduction

✅ Reduce Cookie Banners

Firefox automatically rejects cookie requests on sites that show you that "Can i use your cookies". This only helps to improve your privacy.

Logins and Passwords

🔘 Ask to save logins and passwords for websites

Firefox's inbuilt password manager is secure and encrypted. Mozilla cannot see your passwords. You can learn more about this here. If you use another password manager, t urn this off

✅ Show alerts about passwords for breached websites

If you use Firefox's password manager, be sure to turn on this setting. Basically, it checks known website breaches and if the date that you saved a password for your website is before the breach happened, Firefox will tell you to change your password. The breach data is provided by haveibeenpwned, a trustworthy site.

✅ Use a Primary Password

If you use Firefox's password manager, be sure to turn on this setting. This makes it so you need a password to enter your logins so someone on your PC cant access it, like a family member.

🔘 Allow Windows single sign-on for Microsoft, work, and school accounts

If your school or work makes you turn this on, then do it. But if you need to use a school or work, it's best to use another firefox profile or a different browser entirely to keep them seperated.

History

Read

History cannot be accessed by websites. It is simply something that is stored on your pc. Only clear history if you have to hide it from someone.

Address Bar

🔘 All

This enables different ways for your address bar to be more personalized. No data will ever be sent. Either leave alone or customize if you don't like.

Permissions

Location, Camera, Microphone, Notifications, Autoplay, Virtual Reality

All of these are set to "Always Ask" by default. If you Set these to "Block", you are opening yourself up to device fingerprinting. Keep all of these settings by default.

✅ Block pop-up windows

Some websites like to open popups that are usually malicious. If theres a site that is trustworthy, you can open the popup on that site.

✅ Warn you when websites try to install add-ons

Stops malware sites from installing malicious addons/extensions into your browser.

Firefox Data Collection and Use

Before you turn these off:

This is data that gets sent to Mozilla to help improve Firefox. If you use Firefox Beta/Developer/Nightly, leave it on to help Firefox. The data gets used by developers to help develop Firefox. Mozilla does not sell this data to any third-party from what i know. If you want to help Mozilla develop Firefox, leave it on. If you don't want to send data, turn it off.

Deceptive Content and Dangerous Software Protection

✅ Block dangerous and deceptive content

These settings use Google Safe Browsing to block malware that Google have listed in their database. If you don't want to connect to Google's servers, leave this off. You have to choose whether you want Privacy or Security.

Certificates

✅ Query OCSP responder servers to confirm the current validity of certificates

This is another Security or Privacy scenario. In order to check that a website hasn't been tampered with, Firefox will send data to the certificate authority of that website to check if the site is valid. This means that you will be pinging all CA's constantly.

HTTPS-Only Mode

✅ Enable HTTPS-Only Mode in all windows

Literally nearly every site uses HTTPS, and if it doesn't you'll get a warning. HTTP is unsecure and anyone can see what you're doing on that website. With HTTPS, they can only see the domain you're on. i.e. if you're searching on google.com, they can only see you're on google.com. But if you're using the HTTP version, they can see exactly what you searched.

🔁 Sync

If you have multiple devices then Sign up for a Firefox account. This allows you to sync your passwords, history, bookmarks and else between devices. Mozilla cannot see any of your data in your account.

⚠️ Advanced (about:config)

This can help you tweak Firefox a bit more but heed the warning Firefox gives you. You can access this by entering about:config in the address bar.

The amount of customizations you can do to Firefox with about:config tweaks alone is gigantic and I'll only show a few different user.js files or about:config tweaks.

arkenfox user.js - This is a template file that hardens Firefox for security, privacy and anti-tracking features. I will reiterate that this is a template and it is advised to read the wiki and look through user.js file itself.

r/Firefox about:config tweaks - A bunch of general use customizations

🧩 Extensions

  • uBlock Origin - This is basically a "content blocker". It blocks ads on websites, it blocks malicious websites, it blocks annoying popups, it blocks various types of tracking and makes your browser way faster.
    • uBlock Origin Configuration


    Open the uBlock Dashboard. You can do this by clicking on the icon on the toolbar and clicking the gears.

    If you use Firefox Sync, Enable cloud storage support. Next, go to the Filter Lists category and check the following:

    • - AdGuard URL Tracking Protection
    • - Block Outsider Intrustion into LAN
    • - Phishing URL Blocklist
    • - PUP Domains Blocklist
    • - uBlock filters - Annoyances

    Click Import... and paste https://raw.githubusercontent.com/DandelionSprout/adfilt/master/LegitimateURLShortener.txt into it. Click apply changes and then "Update now"


    If you want to block certain domains dynamically and turn uBlock Origin into a firewall, you can enable "I am an advanced user" and then follow this guide here.

  • SponsorBlock - This automatically skips Sponsorships on YouTube. You can also make it skip self promotion, interaction reminders, intros, non music sections and recaps.
  • Return YouTube Dislike - Returns the YouTube dislike.
  • FastForward - You know those adf.ly and ouo.io links that make you wait 5 seconds before contuining? This extension skips those.
  • Dark Reader - If you like dark mode but some sides are pure white (like Wikipedia), this extension turns them dark.
  • Bypass Paywalls Clean - This has been removed from the Firefox addons store for "copyright" reasons but this bypasses paywalls from websites that require you to pay to see an article.
  • Skip Redirect - Some sites like to add a tracking referrer before directing you to a website. This disables that.

Extensions to avoid

  • I Don't Care about Cookies - This was recently acquired by Avast which is known to be scummy. Use I still don't care about cookies, an open source contuination of the former.
  • Privacy Badger - This extension is practically useless as it "learns" to block trackers. So it doesn't block them all until its "learnt" it. uBlock Origin blocks these by default.
  • HTTPS Everywhere - Redudant with HTTPS Only Mode
  • Decentraleyes/LocalCDN - if you use Strict tracking protection, third party sites are already partitioned and isolated.If you care about not connecting to certain CDN domains, use a VPN or TOR instead.
  • Any other adblockers (AdBlock, AdBlock Plus) - uBlock Origin is already the best adblocker and a lot of these don't have the features uBlock Origin enables.
  • ClearURLs - if you set-up URL tracking protection in uBlock Origin, this extension is redudant.

Other things to look into are Firefox UI Fix if you want a new firefox GUI or want to revert back to the old square GUI (read the Wiki for further customization), or look into the Firefox Gnome Theme if you use GNOME or like Adwaita. You can also look at installing new themes.
